# Email Logs

A lightweight single-file web app to track ticket IDs, timestamps, and status updates (`Resolved` / `On Hold`) and export the log as CSV.

## Project File

- `email-log-vercel.html` - complete app (UI, logic, CSV export)

## Features

- Add ticket entries with timestamp and status
- Toggle status directly from the log table
- Remove entries from the list
- View live totals for overall, resolved, and on-hold items
- Export all records to CSV

## Run Locally

1. Open `email-log-vercel.html` in any modern browser.
2. Start adding ticket IDs and status values.
3. Use **Extract CSV** to download the log.
